📱 Big Day App

O Big Day é um aplicativo desenvolvido em Swift com foco em produtividade pessoal. Ele foi criado para ajudar os usuários a tornarem um dia comum em um Big Day — um dia produtivo, leve e com propósito.

✨ Funcionalidades

✅ Criação de lista de tarefas personalizadas para o dia
🧠 Registro de histórico dos dias concluídos com sucesso
🎨 Personalização dinâmica do app:
Apelido do usuário
Cores do aplicativo
Foto de perfil
🔐 Autenticação com Firebase:
E-mail e senha
Login com Apple ID
Login com Google
📊 Armazenamento de dados no Firebase (Firestore)
💡 Objetivo

Este projeto foi idealizado para ser um marco na minha transição de carreira como desenvolvedor iOS, demonstrando meus conhecimentos em:

Arquitetura MVVM
Integração com Firebase
Personalização de UI/UX
Animações e interações dinâmicas
🛠️ Tecnologias e Ferramentas

Swift + UIKit
Arquitetura MVVM
Firebase Authentication
Firestore
Auto Layout
Swift Package Manager
🚧 Em desenvolvimento

O app está em constante evolução, e novas funcionalidades estão sendo planejadas para melhorar a experiência do usuário.

📱 Big Day App (English)

Big Day is an app developed in Swift focused on helping users turn an ordinary day into a Big Day — a productive, meaningful, and intentional day.

✨ Features

✅ Create custom to-do lists for the day
🧠 Track and store a history of successfully completed days
🎨 Dynamic customization options:
User nickname
App color themes
Profile picture
🔐 Firebase Authentication:
Email and password login
Apple ID login
Google login
📊 Data storage via Firebase Firestore
